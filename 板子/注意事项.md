### Floyd 新加一条边怎么更新

```cpp
int w; cin >> w; //u->v（v->u) = w
if (w >= g[u][v]) continue;
g[u][v] = g[v][u] = w;
rep(i, 1, n) {
    rep(j, 1, n) {
        chkmin(g[i][j], g[i][u] + g[u][j]);
    }
}
rep(i, 1, n) {
    rep(j, 1, n) {
        chkmin(g[i][j], g[i][v] + g[v][j]);
    }
}
```

### 子集和怎么求 $O(2^n \times n)$

```cpp
// sum[i]表示i的所有子集的状态和，同理可以应用于max min
for (int j = 0; j < 20; j ++ ) {
  for (int i = 0; i < 1 << 20; i ++ ) if (i >> j & 1) {
    sum[i] += sum[i ^ (1 << j)];
  }
}
```
